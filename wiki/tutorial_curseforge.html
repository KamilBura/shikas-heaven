Tutorial Installing Modpack through CurseForge
At First you will need to go on the Modpack Webiste. You can choose two Options BASE or EXTENDED.
After choosing Version, it should open in CurseForge and install it.
You can also Download it through Search Page on CurseForge, but it will Install only the Newest Version.
When everything went well, you should see bunch of Mods in mods section. Now just run the Game and have Fun!
