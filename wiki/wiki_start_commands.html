/c or /class => to choose class

/p, /player or SNEAK + RIGHT CLICK => to see Player Informations

/att or /attribute => to see your Attributes list

/mmobars layout set {your name} default / layout_soul-like / layout_wide => to change your Health Bar Layout

/afk => to set yourself AFK or UNAFK yourself

/rtp => to teleport yourself to Random Location

/r or /reply => to Reply to an MSG

/banhistory => to see History of Banned People

/chatchannel => to change chat Channel for example Trade channel

/playerlist => to see all Online Players

/tell => to send someone Private Message

/chairs => to sit on a Half slabs

/skin set {name of a player} => to set a Skin

/ecogui => to see all new Enchants and Descriptions of them

/ecoenchants toggledescriptions => toggle a Descriptions under Enchants of an Item

/ecoenchants enchantinfo {enchant name} => Tell you about Enchant
